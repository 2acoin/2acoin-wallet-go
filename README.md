# TurtleCoin Nest

## Installation

[release 0.10-beta-linux](https://github.com/woodyjon/TurtleCoin-Nest/releases/tag/0.10-beta)

Install this binding: https://github.com/therecipe/qt and run `qtdeploy build desktop`
